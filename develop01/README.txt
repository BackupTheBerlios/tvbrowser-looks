Developer Info
==============
This project is currently developed with Eclipse 3.1 - feel free 
to use any tool you like but be informed that the easiest way to
build it is with Eclipse 3.1, alternativly use ANT (available at
http://ant.apache.org/).

When building with ANT, the default target build the project jar 
and/or distribution files into the directory "build/target".
